# Mi Lan Font Module

This Magisk module provides Mi Lan fonts for your device

![](https://github.com/WSTxda/Mi-Lan-Font-Module/blob/bdb0b3d39c968b2419dd4fc1be4b0e07deb7617a/files/banner.png)
 
## Requirements 
- Android 7.0+ 
- Magisk 20+ 

## Installation 
1. Flash this module in Magisk Manager
2. Reboot device
3. Enjoy!
