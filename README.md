# Mi Lan Font Module

This Magisk module provides MIUI fonts for your device

![](https://i.imgur.com/cKSrKFB.png)
 
## Requirements 
- Android 7.0+ 
- Magisk 20+ 

## Installation 
1. Flash this module in Magisk Manager
2. Reboot device
3. Enjoy!
